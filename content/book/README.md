# The Will of Heroes - Book Manuscript

This folder will contain Colin's book manuscript for the AI Coach to use as its knowledge base.

## Supported Formats

You can add your manuscript in any of these formats:
- `manuscript.txt` - Plain text (recommended for best AI processing)
- `manuscript.md` - Markdown formatted
- Individual chapter files: `chapter-01.md`, `chapter-02.md`, etc.

## How It Will Be Used

The AI Coach will:
1. Read the full manuscript content
2. Extract key concepts, principles, and hero stories
3. Use this knowledge to provide personalized coaching based on YOUR methodology
4. Reference specific quotes and examples from the book

## Adding Your Manuscript

1. Export your book as plain text or markdown
2. Place the file(s) in this folder
3. The system will automatically index the content

If you want to organize by chapter:

## Privacy Note

Your manuscript content is stored locally and only used to train YOUR AI coach.
It is not shared with any third parties or used for any other purpose.
